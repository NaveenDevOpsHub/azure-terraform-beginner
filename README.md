# azure-terraform-beginner
Provision an Azure Resource Group + Storage Account

# What you'll Build
A Resource Group
An Azure Storage Account
Use Terraform CLI to deploy and manage infrastructure

# Project Structure
azure-terraform-beginner/
├── main.tf
├── variables.tf
├── outputs.tf
└── README.md

## Deployment Steps
# Authenticate:
az login
# Initialize Terraform:
terraform init
# Preview changes:
terraform plan
# Apply configuration:
terraform apply

# Learning Outcomes
Understand Terraform syntax and workflow
Practice using variables and outputs
Learn how to manage Azure infrastructure declaratively.
